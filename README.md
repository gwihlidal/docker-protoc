# docker-protoc
Docker image with protobuf compiler and various language plugins

## Usage
```
$ docker run --rm gwihlidal/protoc --help
Usage: /usr/bin/protoc [OPTION] PROTO_FILES
```

## Supported languages and tools
- C
- C++
- C#
- Rust
- Python
- Ruby
- Go
- Java / JavaNano
- JavaScript
- Objective-C
- Swift 4
- Swagger
- grpc-gateway
